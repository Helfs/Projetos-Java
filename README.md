Projetos-Java
=============

Todos projetos criados em java.

Interdisciplinary project created on July, 2013 for the Project Applied for Programmers.
It was requested to create a Real State system capable of register a client, an enployee and a house using NetBeans
integrated with Hibernate and JDBC as database.

This system was created by a group of 3 people where I was responsilble to create the house registering part.
